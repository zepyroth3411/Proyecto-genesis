Control Remoto para un Futuro Autónomo

  • Objetivo del proyecto: Demostrar como la tecnologia puede superar barreras fisicas
  • Facilitar el control remoto de dispositivos complejos.
  • Resaltar la importancia de la comunicación cliente-servidor en aplicaciones prácticas.
  • Integrar interfaces web intuitivas con tecnología avanzada para resolver problemas reales.

Este proyecto surge como parte de una materia universitaria, donde el reto era diseñar un sistema innovador de control remoto. La solución permitió a una persona, desde un aula, enviar comandos a través de otro módulo de control, para mover un brazo robótico ubicado en otro espacio. Todo el proceso se gestiona con microcontroladores, demostrando cómo la tecnología puede superar las barreras físicas.
